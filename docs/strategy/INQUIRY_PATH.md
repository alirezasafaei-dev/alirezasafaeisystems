# Inquiry Path — Professional Review & Collaboration

**Status:** Active (E1-05)  
**Last Updated:** 2026-07-06

---

## Purpose

Give visitors a clear, honest path to request technical work **without** pretending there is a large company or open hiring program.

**Primary product path remains ASDEV Audit** — self-serve assessment before custom work.

---

## Routes

| Route | Use |
|---|---|
| https://audit.alirezasafaeisystems.ir/sample-report | See report format and trust signals |
| https://audit.alirezasafaeisystems.ir/audit | Start free website assessment |
| `/qualification` (this site) | Scoped professional review + quick-fix inquiry |

---

## What we accept

- Professional website audit follow-up (after ASDEV Audit report)
- Technical SEO review for a specific live URL
- Performance or security review with defined scope
- Small fixed-scope implementation sprints (when capacity allows)
- Collaboration inquiries with clear problem statement and URL

---

## What we do not accept

- Open-ended "build my app" without scope
- Fake hiring / bulk role applications (no active hiring unless owner announces)
- Work on frozen ASDEV products (DevAtlas standalone, unrelated SaaS)
- Guarantees on rankings, revenue, or uptime without a signed scope
- Requests without a reachable public URL (for site-related work)

---

## Recommended flow

1. Run ASDEV Audit on the live site (or review sample report).
2. If human execution is needed, submit `/qualification` with URL, main issue, and contact channel.
3. Initial response targets whether a **fixed-scope** sprint is appropriate — not an open retainer promise.

---

## Wording (use these, not "hiring")

- Professional review request
- Project inquiry
- Audit collaboration
- Request technical review

---

## Operational limits (honest)

- Solo-operator capacity — response time goal: one business day for first triage (not SLA-guaranteed)
- Evidence pending for public inquiry volume metrics
- Payment and contracts only after scope agreement (billing activation is separate roadmap phase)
